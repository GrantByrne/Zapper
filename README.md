# Zapper


**Important** - This project is currently a work in progress. Please don't try to use it. I'm certain you will be quite disapointed.


## Overview


The aim of the Zapper project is to create a unversal remote hub that runs on a Raspberry Pi.


## Doesn't That Basically Already Exist?

In one form or another, yes. Prior to starting this project my main remote hub was the Logitech Harmony Hub with remote. [Unfortunately, Logitech is dropping the Harmony lineup.](https://www.tomsguide.com/news/logitech-kills-off-harmony-universal-remotes-but-theres-a-little-good-news) 

In addition to this, I've been happy-ish with my current setup; however, I would like to utilize more control over my home theater. I would like a set up that more seemless handles my amplifier, better switches between inputs, and interacts more directly with my LG TV running Web OS.

I also think that since home theater set up can get really complicated with many many different components, it's a problem that lends itself well to crowd sourcing through open source.

My hope is to make a system that meets my needs, but it also extensible to fits the needs of other people looking to have a more flexible universal remote experience.
